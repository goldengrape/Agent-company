# 新公司创建需求：官僚式小说创作公司 (Bureaucratic Fiction Writing Company)

## 业务领域
小说创作（长篇、中篇），采用专业化分工的"官僚式"创作流程，确保作品质量。

## 核心理念
将小说创作流程拆解为多个专业岗位，通过严格的文件流转和审计机制保证创作质量——就像一个高效运转的官僚机构。

## 需要的岗位角色

### 创作管理类
1. **创作总监** — 统筹全局，维护核心设定，审批重大修改
2. **故事建筑师** — 负责故事结构设计（三/四幕结构、场景排列）

### 角色与对白类
3. **角色设计师** — 设计人物核心卡、卡司配置、人物弧线
4. **对白大师** — 编写对白、建立角色声音库、审查潜文本

### 内容创作类
5. **场景导演** — 负责具体章节/场景的撰写
6. **符号织造者** — 管理母题库、隐喻网络、象征系统

### 审查类
7. **结构审计员** — 审查故事结构、场景价值变化、节奏逻辑
8. **历史考据官** — 审查历史准确性（适用于历史题材）
9. **语言审查官** — 审查情感展示（Show Don't Tell）、修辞质量

## 主要工作流程

### 创作流程（PDCA）
1. **计划阶段**：创作总监发布创作任务 → 故事建筑师设计结构蓝图
2. **执行阶段**：角色设计师设计人物 → 场景导演撰写章节 → 对白大师润色对白
3. **检查阶段**：结构审计员 + 历史考据官 + 语言审查官分别审查
4. **处理阶段**：创作总监根据审查结果决定通过或返工

## 核心公文类型
- **创作任务书** — 包含故事主题、基调、目标读者
- **结构蓝图** — 三/四幕结构、场景列表、节奏设计
- **人物核心卡** — 每个角色的完整设定
- **角色声音库** — 角色对白风格指南
- **章节初稿** — 场景导演撰写的章节
- **审计报告** — 结构审计、对白诊断、情感审查、历史考据等报告
- **核心设定修正日志** — 追踪所有设定变更
